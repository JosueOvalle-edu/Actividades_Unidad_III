Preposiciones Lógicas



1\. Obtener el certificado de inglés.



&nbsp; Condiciones: Estudiar el vocabulario y aprobar el examen oral.

&nbsp; 

&nbsp; Explicación: El vocabulario permite entender las palabras, y el examen oral es para demostrar la fluidez, así que, ambos son necesarios para obtener el certificado.



&nbsp; Tipo: Conjunción (Y)



&nbsp; Enunciado: Si estudio el vocabulario **Y** apruebo el examen oral **ENTONCES** obtendré el certificado de inglés. 



2\. Poder ver una película en el cine.



&nbsp; Condiciones: Comprar las entradas por medio de la aplicación o tener una tarjeta de membresía.



&nbsp; Explicación: Cualquiera de las dos opciones permite el ingreso a la sala de cine, así que, solo uno es necesario para poder ver una película en l cine.



&nbsp; Tipo: Disyunción (O)



&nbsp; Enunciado: Si compro las entradas en la aplicación **O** tengo una tarjeta de membresía **ENTONCES** podré ver la película.



3\. Comprar una computadora nueva.



&nbsp; Condiciones: Tener el dinero suficiente ahorrado y no tener deudas en la tarjeta.



&nbsp; Explicación: El ahorro asegura la compra y que no tenga deudas asegura la estabilidad financiera a largo plazo. Ambos son necesarios para poder comprar una computadora nueva.



&nbsp; Tipo: Conjunción (Y)



&nbsp; Negación: No tener deudas es la negación de tener deudas.



&nbsp; Enunciado: Si tengo el dinero suficiente ahorrado **Y NO** tengo deudas **ENTONCES** podré comprarme una nueva computadora.



4\. Hacer ejercicio en la madrugada.



&nbsp; Condiciones: Madrugar y que el gimnasio abra temprano.



&nbsp; Explicación: Levantarse temprano asegura la disponibilidad para hacer ejercicio y que el gimnasio abra temprano asegura poder ir a esa hora. Ambos necesitan ser verdaderos para poder hacer ejercicio en la madrugada.



&nbsp; Tipo: Conjunción (Y)



&nbsp; Enunciado: Si me levanto temprano **Y** el gimnasio abre temprano **ENTONCES** podré hacer ejercicio en la madrugada.



5\. Ver un partido de fútbol.



&nbsp; Condiciones: Tener una suscripción a Disney+ o tener contratado televisión por cable.



&nbsp; Explicación: Cualquiera de las dos condiciones permitirá ver el partido, solo uno es necesario para poder ver un partido de fútbol.



&nbsp; Tipo: Disyunción (O).



&nbsp; Enunciado: Si tengo una suscripción a Disney+ **O** tengo contratado televisión por cable **ENTONCES** podré ver el partido de fútbol.
